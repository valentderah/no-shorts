# No Shorts

Браузерное расширение для блокировки коротких видео на YouTube, TikTok, VK и Instagram.

## Возможности

- **Блокировка страниц**: Блокирует доступ к разделам коротких видео с помощью кастомной заглушки
- **Очистка навигации**: Скрывает элементы навигации, ведущие к коротким видео
- **Очистка ленты**: Удаляет рекомендации коротких видео из основных лент
- **Поддержка платформ**: YouTube, TikTok, VK, Instagram
- **Настройки**: Включение/выключение блокировки для каждой платформы отдельно

## Установка

### Ручная установка

1. Клонируйте репозиторий
2. Установите зависимости:
   ```bash
   npm install
   ```
3. Соберите расширение:
   ```bash
   npm run build
   ```
4. Откройте страницу расширений Chrome/Edge (`chrome://extensions`)
5. Включите "Режим разработчика"
6. Нажмите "Загрузить распакованное расширение" и выберите папку `dist`

### Разработка

```bash
# Режим разработки с отслеживанием изменений
npm run dev

# Сборка для продакшена
npm run build

# Очистка директории сборки
npm run clean
```
